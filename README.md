# Setup
1. Install Python3
- Termux
```bash
pkg update && pkg upgrade -y
pkg install python python2 python3 -y
python --version
python3 --version
```
- iSH
```bash
apk add python3
apk add py3-pip
```
###
2. Downloading & Running crash.py
```bash
git clone https://github.com/0725bubz-art/Termux-and-iSH-crash
cd Termux-and-iSH-crash
python3 crash.py
```
